# 3.3 厂商的目标
`首先如何计算利润？企业谋求超额利润，完全竞争时的企业只能获得正常利润；之后三种思路谋求利润最大化：总成本、总收益、边际收益思路；MC = MR 思路；MRP = P 思路；`

## 利润计量方式
1. 会计利润：经济利润 + 正常利润；
2. 经济利润（超额利润）：总收益 - (显性／实际应用成本 + 隐性／边际成本)；
3. 正常利润：抵消掉隐性／机会成本的会计利润

经济利润 = 0，会计利润等于正常利润；经济利润 > 0，企业赚取**超额利润**；经济利润 < 0，企业蒙受**经济损失**；

※ 当市场处于均衡状态时，经济利润等于零（在完全竞争市场，一旦企业可以获得超额利润，其他企业将会立刻进入市场，长期还会回到正常利润）

### 经济租
所有**超过机会成本**的**支出**就是经济租。

## 企业利润最大化
1. 总收益（TR）减总成本（TC）差值最大时；边际产出（MP）先 ↑ 后 ↓；当 MP 到达峰值，
2. **MR = MC**；当企业无法通过调整产量获取最大收益时，即达到最优化产量（利润最大化）；
3. MRP（边际收益产品，<img src="http://latex.codecogs.com/png.latex?MP \times MR">）

※ MC 曲线交 ATC 曲线于最低点（**盈亏平衡点**，breakeven point）；交 AVC 曲线于最低点（**关停点**，shutdown point）

### 规模经济／规模不经济（长期企业利润最大化）
（**第二种利润最大化思路**）**SRATC 曲线分析**，短期内规模无法改变，企业只能寻求 MR = MC；
（**第一种利润最大化思路**）**LRATC 曲线分析**，变企业规模小时，规模 ↑ 效率 ↑（分工）：规模经济；企业规模大时，规模 ↑ 效率 ↓； 中等规模，规模收益恒定；**使得LRATC 曲线到达最低点的（区间中的）最低产量，最低有效规模点（munimum efficient scale）**；

### 长期供给曲线（3 种）
1. 厂商数量 ↑，要素价格 ↑，成本增加型市场（increasing-cost industry）；
2. 厂商数量 ↑，要素价格 ↓，成本降低型市场（decreasing-cost industry）；
3. 要素价格不随产量变化，成本均衡型市场（constant-cost industry）;

### 生产率
MP 最高值对应 MC 的最低值；
AP 最高值对应 AVC 的最低值；
（**第三种利润最大化思路**）<img src="http://latex.codecogs.com/png.latex?MRP_{i}=P_{i}"> 时企业利润最大化；① MRP > P 时，要素生产率 > 成本，企业会多使用这个要素；② MRP < P 时，生产率 < 成本，企业会减少使用这个要素；③ **MRP = P 时，企业利润最大化**；
