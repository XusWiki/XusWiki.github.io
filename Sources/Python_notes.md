## 4.1 Python 程序的结构
一个典型的Python 程序可以被分解为：模块、语句、表达式、对象。其中：

* 对象 ∈ 表达式；
* 表达式 ∈ 语句；
* 语句 ∈ 模块；
* 模块 ∈ 程序

亦即，符合以下包含关系：“**程序（模块（语句（表达式（对象））））**”。

## 4.2 核心数据类型
Python 可以采用以下对象类型。对象类型类似“容器”这个概念，是用以存储不同类型的数据的单元。

|对象类型|范例|
|===|===|
|数字|`1234`，`3.1415`，`3 + 4j`，小数，分数|
|字符串|`'spam'`，`"guido's"`，`'''good taste'''`，`b'a\xolc`|
|列表|`[1, [2, 'three'], 4]`|
|字典|`{'food':'spam', 'taste':'yum'}`|
|元组|`(1, 'spam', 4, 'U')`|
|文件|`myfile = open('eggs', 'r')`|
|集合|`set('abc'), {'a', 'b', 'c'}`|
|其他类型|类型、Nome、布尔型（用“1”和“0”表示“true”或“false”）|
|编程单元类型|函数、模块、类|
|与现实相关的类型|编译代码堆栈跟踪|

### 4.2.1 数字

可以输入的数字类型：

- 整数；
- 浮点数（带有小数点的数字）；
- 含有虚数部分的复数；
- 十进制数；
- 有理数（包含分子和分母）；
- 集合

可以对数字进行的操作：

- `+`：加法；
- `-`：减法；
- `*`：乘法；
- `/`：除法；
- `**`：乘方

另外也可以导入模块进行数学运算：

`>>> import math` 导入 Python 数学模块

`>>> math.pi`

`3.141592658973`

`>>> math.sqrt(81)` 开平方根

`9`

`>>> import random` 导入随机数模块

`>>> random.random()`

`0.7975225822640719`

`>>> random.choice([1, 2, 3, 4, 5, 6])`

`3`

### 4.2.2 字符串

在 Python 中，字符串是被作为序列处理的。换言之，字符串中的每一个字符，都有其序号。

`>>> Str = 'Spam'`

`>>> len(Str)`

`4`

`>>> Str[0]` 索引字符串的第一个字符

`S`

`>>> Str[1]`

`p`

`>>> Str[-1]` 索引字符串的最后一个字符

`'m'`

用户也可以对字符串进行操作：

`>>> Str + 'xyz'`

`Spamxyz`

`>>> Str * 2`

`SpamSpam`

注意: 当一个字符串被定义后便无法再被修改。想要更改这个字符串的内容，用户需要重新定义这个字符串。

`>>> Str = 'z' + Str[1:]` 重新定义字符串

`>>> zpam`

笔记: 以上操作也可以用于**列表**以及**元组**。字符串还有其特殊的操作形式（Special-Type Methods），这些操作形式无法被其他类型的变量所使用。详见下。

① 查找字符序号的 *find* 功能：

`>>> Str.find('pa')`

`1`

② 对字符串进行全局搜索和替换的 *replace* 功能：

`>>> Str.replace('pa', 'xyz'`

`Sxyzm`

③ 将字符串拆分为下级字符串的 *split* 功能：

`>>> Str = 'S, p, a, m'`

`>>> Str.split(',')`

`[S, p, a, m]`

④ 将字符串进行大小写变换的 *upper* 和 *lower* 等功能：

`>>> Str = 'i\'m good!'`

`>>> Str.upper()` 转大写

`'I'M GOOD'`

`>>> Str = Str.upper`

`>>> Str.lower()` 转小写

笔记: 其余还有大小写互换：`Str.capitalize() `；首字母大写：`Str.title()` 等功能。

⑤ 测试字符串内容的 *isalpha* 以及 *isdigit* 功能：

`>>> Str.isalpha()`

`True`

`>>> Str.isdigit()`

`False`

⑥ 去除字符串结尾空格字符的 *rstrip* 功能：

`>>> Str = 'A & B `

`>>> Str.rstrip()`

`>>> 'A & B'`

⑦ 替换内容的 *format* 功能：

`>>> '{0} hate {1}.'.format('Cats', 'dogs')` 2.7 版 Python 的写法

`>>> 'Cats hate dogs.'`

`>>> {} hate {}..format('Cats', 'dogs')` 3.5 版 Python 的写法
