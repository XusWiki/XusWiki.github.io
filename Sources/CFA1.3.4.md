# 3.4 厂商与市场结构
## 四种市场
|:---:|:---:|
|① 完全竞争|② 垄断竞争|
|③ 寡头垄断|④ 完全垄断|

通过有几种因素考虑市场结构：
- 厂商数量：① > ② > ③ > ④；
- 产品差异度：① < ② < ③ < ④；
- 价格控制：① > ② > ③ > ④；
- 入市门槛：① < ② < ③ < ④；
- 非价格竞争：① < ② < ③ < ④

### 完全竞争市场
长期看只有正常利润（因为入市门槛低，一旦市场有经济利润，便会有新竞争者进入；超额利润因厂商进入额日趋进于 0）。

### 垄断竞争市场
相比**完全竞争市场**，垄断竞争市场反应较慢（在**长期**不存在经济利润，短期可能存在）；产品开始出现差异，当时还是可以被替代；
相比**寡头垄断市场**，寡头垄断市场壁垒更高，议价能力更强，更难进入。

### 寡头垄断市场
产品依然存在同质化，但是差异更大；壁垒；
四种分析模型：
1. 拐折需求曲线模型：厂商之间“跟跌不跟涨” → 一旦价格提升，消费者会转向其他厂商 → 提价弹性大，降价弹性小；
2. 古诺双寡头模型；
3. 纳什均衡；
4. 主导厂商模型：主导厂商决定价格，跟随着根据价格决定产量。

### 完全垄断市场
只有一个厂商（高度差异化：无替代品）；进入壁垒极高；厂商议价能力极强；最大利润产量依然符合 MC = MR;
两种价格决定方法：
1. 单一价格决定法：MC = MR；P > ATC；
2. 价格歧视：为了**尽量攫取消费者剩余**，并将其转化为经济利润额(e.g. 参观展览学生票)；价格歧视需要满足三个个假设：① ↘ 需求曲线；② 至少两组消费者；③ 不存在消费者之间转让商品；

自然垄断市场（natural monopoly）：某一企业能够以低成本（价格）提供某一商品时，其他企业无法生存；

平均成本定价法：垄断厂商不追求最高价格，而追求最高利润；当采取 MC = MR 时（无政府监管）存在无谓损失；监管：平均成本定价法／公平回报定价法（average cost pricing / fair return pricing）；P 取 ATC 交需求曲线点；

边际成本定价法（margin cost pricing）

### 市场集中度（计算企业议价能力）
1. 四企业集中度：当四企业集中度超过 40% 时，被认为是寡头垄断市场；当四企业集中度低于 40% 时，被认为是垄断竞争市场；
2. Herfindahl-Hirschman Index, HHI：市场占有率的平方之和；相较于四企业集中度，HHI 对于公司合并影响比较敏感；① 竞争性行业，HHI < 1000；② 适度竞争行业，1000 < HHI < 1800；③ 非竞争性行业，1800 < HHI < 10000；④ 完全垄断 HHI = 10000；
