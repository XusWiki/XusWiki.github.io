# 3.1 导引
市场：①**商品市场**（公司 *business firms* → 家庭 *householder*）；②**要素市场**（资本、劳务、素材；家庭 → 公司）

## 供给、需求与市场均衡
- 在供给／需求曲线上点的移动：价格变化；
- 供给／需求曲线的移动：其他因素变化（生产力、收入水平 etc）

### 曲线
曲线上有 P、Q 两个因素；市场均衡 → 供给／需求曲线交点的 P 就是**均衡价格**；对于同一个 P：
- QS > QD：供给过剩（excess supply）；
- QD > QS：需求过剩（excess demand）

### 供求定理
边际（margin）：
- 边际收益（margin benefit，MB）：Q 每增加 1 个单位所增加的收益，边际收益递减 → **MB 曲线也就是需求曲线**；
- 边际成本（margin cost，MC）：Q 每增加 1 个单位所增加的收益，边际成本递增 → **MC 曲线也就是供给曲线**

※ 类似供给需求曲线的原理，当 MC = MB 时达到均衡，商品数量有效，**消费者剩余（consumer surplus）、生产者剩余（producer surplus）达到最大化**；当 MC > MB，企业可以通过降低产量（↓）来提高收益；当 MC < MB，且也可以通过提高产量（↑）来提高收益

### 稳定均衡与不稳定均衡
- 稳定均衡：偏离均衡点 → 重新回到均衡点 → 均衡点；
- 不稳定均衡：偏离均衡点 → 难以返回均衡点 → 不稳定均衡

## 拍卖
自由市场是一种达成均衡价格的方式，拍卖（auction）另一种方式。

叫价形式区别：
1. 英式拍卖；
2. 密封拍卖；
3. 荷兰式拍卖；
4. 单一价格拍卖

偿价形式区别：
1. 共同价值拍卖（common value auction）：每个投标人付费相同；
2. 个人价值拍卖（private value auction）：每个投标人付自己的定价

## 消费者剩余、生产者剩余及无谓损失
<center>
![](https://upload.wikimedia.org/wikipedia/commons/d/d7/Economic-surpluses.svg)
</center>

- 消费者剩余（consumer surplus，CS）：消费者*愿意付出**的价格和**实际付出**的价格的差值；SΔMBP；
- 生产者剩余（producer surplus，PS）：生产者**实际收到**的价格和**最低接受**的价格的差值；SΔMCP

当达到有效产量，CS 与 PS 都达到最大，并且二者之和亦达到最大；如果未能达到最优，说明存在无谓损失（deadweight loss，DWL）：
- 如果 Q > Q*：是产量过剩造成的 DWL；
- 如果 Q < Q*：是产量不足造成的 DWL

造成 DWL 上升的其他因素：
- 价格控制（price ceiling）；
- 价格下限；
- 税（税收由买卖双方共同承担，议价能力较弱的一方承担较大的比例／税收负担更多低落在缺乏弹性的一方）；
- 补贴；
- 配额

## 弹性
需求价格弹性：因为价格（P）变动造成的需求量（Q）变动；是一个比例，变动百分比的比率
<center><img src="http://latex.codecogs.com/png.latex?\varepsilon_{d}=\frac{\%\Delta Q}{\%\Delta P}=\frac{(Q_{1}-Q_{0})/(Q_{1}-Q_{0})/2}{(P_{1}-P_{0})/(P_{1}-P_{0})/2}"></center>

1. <img src="http://latex.codecogs.com/png.latex?|\varepsilon_{d}|<1"> 缺乏弹性；
2. <img src="http://latex.codecogs.com/png.latex?|\varepsilon_{d}|=1"> 单位弹性；
3. <img src="http://latex.codecogs.com/png.latex?|\varepsilon_{d}|>1"> 富有弹性；
4. <img src="http://latex.codecogs.com/png.latex?|\varepsilon_{d}|=\infty"> 完全弹性；
5. <img src="http://latex.codecogs.com/png.latex?|\varepsilon_{d}|=0"> 完全无弹性

※ 同一条需求曲线上的**每一点**上的弹性**都不相同**，在单位弹性点上，总收益（TR）最大；偏离单位弹性点，价格上升（↑）弹性增加（↑）总收益减小（↓），价格下降（↓）弹性下降（↓）总收益减小（↓）

需求收入弹性：因为收入水平（I）变动造成的需求量（Q）变动；是一个比例，变动百分比的比率
<center><img src="http://latex.codecogs.com/png.latex?\varepsilon_{I}=\frac{\Delta Q/Q}{\Delta I/I}=\frac{\Delta Q}{\Delta I}\times\frac{I}{Q}"></center>
※ 不同商品的需求收入弹性：
- 正常品：（ε > 0）收入提高，购买增加；
	- 奢侈品:（ε > 1）
	- 必需品:（0< ε <1）
- 劣等品：（ε < 0）

需求交叉弹性：一种商品价格变化对另一种商品的影响
<center><img src="http://latex.codecogs.com/png.latex?\varepsilon_{c}=\frac{\Delta Q/ Q_{A}}{\Delta P_{B}/P_{B}}=\frac{\Delta Q_{A}}{\Delta P_{B}}\times\frac{P_{B}}{Q_{A}}"></center>
※ 不同商品的需求交叉弹性：
- 互补品：（ε < 0）；
- 替代品：（ε > 0）
