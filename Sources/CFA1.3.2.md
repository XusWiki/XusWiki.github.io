# 3.2 消费者选择理论
影响消费者选择的 2 个因素：
1. 消费意愿——“消费者偏好” → **无差异曲线**；
2. 财力限制——“预算约束” → **预算约束线**


## 一、消费者偏好
消费服务或商品的满足程度 → **效用**（utility）；所有产生相等效用的点的集合 → **无差异曲线**；无差异曲线的**斜率** k，等于*消费者愿意用一种商品替代另一种商品的比率*，**边际替代率（MSR）**。

1. 消费者对于较高无差异曲线的偏好 > 对于较低无差异曲线的偏好；
2. 无差异曲线向右下方倾斜；
3. 无差异曲线不可相交；
4. 无差异曲线凸向原点

## 二、预算约束
受**消费者的收入**和**价格**影响 → 所能购买的不同数量的商品组合 → **预算约束线**。

两种移动：
1. 收入变化 → 预算约束线平移 → **收入效应**；
2. 价格变化 → 斜率变化 → **替代效应**

## 最优选择
最优：无差异曲线与预算约束线的切点。

1. 最优点 → 无差异曲线**斜率** = 预算约束线**斜率**；
2. 最优点 → 边际替代率 = 相对价格
